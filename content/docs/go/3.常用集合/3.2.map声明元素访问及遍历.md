# 3.2.map声明元素访问及便利
* 声明
```$xslt
m := map[string]int{"one": 1, "two": 2, "three": 3}

m1 := map[string]int{}

m1["one"] = 1

m2 := make(map[String]int, 10 /*Initial Capacity*/)
```
* Map元素的访问
在访问的key不存在时,仍会返回零值,不能通过返回nil来判断元素是否存在
```$xslt
if v, ok := m1[3]; ok {
	t.Logf("key 3 is %d", v)
} else {
	t.Log("key 3 is existing")
}
```
