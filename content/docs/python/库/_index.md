# 库

## 标准库
### 其中常用的库有
    datetime：日期和时间处理相关
    random：随机取值相关
    josn：json 相关
    XML：XML 相关
    collections：集合相关
    base64：base64 编码相关
    hashlib：摘要算法相关，如 MD5、SHA1
    itertools：迭代工具相关
    contextlib：上下文管理相关
    urllib：HTTP 请求相关
