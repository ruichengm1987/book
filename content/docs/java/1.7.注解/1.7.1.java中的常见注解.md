# 1.7.1.java中的常见注解
## JDK自带注解
* @Override 覆盖父类的方法
* @Deprecated 表示已经过时
* @Suppvisewarnings  @SuppressWarning("deprecation") 忽略deprecation警告

## java第三方注解
### Spring
* @Autowired 
* @Service 
* @Repository

### Mybatis
* @InsertProvider
* @UpdateProvider
* @Options
